# Velocity | Luxury Cars Showroom

A modern, Netflix-style cinematic web application for a luxury car showroom.
Built with **FastAPI**, **Jinja2**, and **Vanilla JavaScript/CSS**.

## Features
- 🎥 Cinematic Netflix-inspired UI & Intro Loader
- 🏎️ Dark theme with Neon Red highlights
- 🎵 Background music control
- 💖 Interactive "Like" functionality with floating hearts
- ✨ Ambient floating particles
- 📱 Fully Responsive Design

## Setup & Run Instructions

1. **Install Dependencies**
   ```bash
   pip install -r requirements.txt
   ```

2. **Run the FastAPI Server**
   ```bash
   uvicorn main:app --reload
   ```

3. **View the Website**
   Open your browser and navigate to:
   [http://127.0.0.1:8000](http://127.0.0.1:8000)

## Placeholders
- **Images:** Unsplash image URLs are used as placeholders in `templates/index.html`. You can replace them by adding files to the `static/images/` folder and updating the `src` attributes.
- **Music:** A placeholder is expected at `static/music/cinematic.mp3`. You should add an mp3 file there to enable background music.
